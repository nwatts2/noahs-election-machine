To run this app, Node must be installed on your system. Instructions are as follows:

1) Open two terminal windows, with both in the root directory

2) Run 'npm install' in the first window

3) Run 'npm run start' in the first window. This will start the server.

4) In the second terminal window, run 'cd client'

5) In the second window, run 'npm install'

6) Finally, in the second window, run 'npm run start'. This will start the React development server.
